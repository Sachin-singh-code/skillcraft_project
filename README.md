# 💻 Web Development Mini Projects

A collection of beginner-to-intermediate frontend projects built using **HTML, CSS, and JavaScript**. These projects demonstrate core concepts of UI design, DOM manipulation, and interactivity.

---

## 📦 Projects Included

### ⏱️ Stopwatch
- Start, Pause, Reset functionality  
- Lap tracking system  
- Clean and simple UI  

---

### ❌⭕ Tic-Tac-Toe
- 2-player interactive game  
- Win/draw detection logic  
- Dynamic UI updates  

---

### 🧮 Calculator
- Basic arithmetic operations (+, −, ×, ÷)  
- Responsive layout  
- Error handling (division by zero)  

---

### 📝 Remindly Pro (Task Manager)
- Add, delete, and complete tasks  
- Local storage support  
- Dark mode toggle  
- Progress tracking UI  

---

### 🌐 Landing Page
- Responsive navigation bar  
- Smooth scrolling  
- Scroll-based navbar effects  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla)  
- Bootstrap (in some projects)  

---

## 🚀 How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/your-repo-name.git
   ```

2. Open any project folder  
3. Run the `index.html` file in your browser
